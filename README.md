# 👁️ Somethingagotchi

Un juego tipo tamagotchi psicológico hecho en Java.

Inspirado en el concepto de Something de OMORI.

---

## 🧠 Descripción
Es un juego de consola donde interactúas con una entidad llamada "Something".
Dependiendo del estado del jugador (cordura, miedo, energía), el comportamiento cambia.

También incluye rutas especiales:
- Sunny
- Mari
- Basil

---

## 💻 Hecho con:
- Java
- IntelliJ IDEA

---

## 🎮 Cómo ejecutar
1. Abrir el proyecto en IntelliJ IDEA
2. Ejecutar Somethingagotchi.java

---
